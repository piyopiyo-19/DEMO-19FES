参加条件・出展可能な作品について

参加条件

  • 宗拓作品を1点以上出展可能であること。
    • ※宗拓を意識して作られた単体グッズなども対象

⭕️出展可能な作品

  • 作品形式不問で自由に出展可能となります。
    （例：漫画 / イラスト / 小説 / グッズ / コスプレ / 動画 / ぬいぐるみ等アート写真の展示 / ネットプリントetc…）
  • 既刊 / 過去SNS等で公開済みの作品
その他出展可否が不明な作品がありましたら、主催まで問い合わせください。

❌️出展が認められない作品

  • 逆CP / リバ要素のある作品
  • 年齢指定作品のネットプリント
  • 委託での出展
  • 自作ではない作品 / AI生成による作品
  • 有償DL販売専用作品
作品表記についての注意事項

作品の取り扱いについて

  • 年齢指定作品(R18 / R18G)は必ず年齢確認をお願いします。
    • ※注意書き・サンプル画像の修正等の配慮をお願いします。
  • 他CP要素が含まれる場合、その旨を注意書き記載ください。
  • 特殊な設定 / 死ネタ等を取り扱う場合、お品書きや作品の該当ページに注意書き記載をお願いします。

出展区分について

出展区分とは

出展区分とはどのサービスを利用し作品公開をするかの設定になります。PICTSQUAREでは下記3つの区分を選択することが可能です。

①pictSQUARE-ONE

PICTSQUARE系列サービスの為、 外部サイトへの移動警告を出さずシームレスに作品への案内ができます。
同人誌やグッズの通販機能のほか、金銭取引のない 作品展示のみを行う機能もあります。

【補足】外部リンク集としての使い方
  • ストア機能から「リンク」を選択することで、外部サイトへのリンク集のような使い方も可能です。
    • 例：作品サンプルをスペース上で表示→購入ページは外部サイト（BOOTH等）へ誘導…など
  • 時間に余裕がない・面倒な事は避けたい方は、③のその他通販サイトを使用→Litlink等のURLにまとめる方法を推奨します。
  • ストア内の文字・装飾はHTML/CSS編集に対応しています。コーディング経験がない方でも比較的簡単にページ作成ができるツールもありますので、必要に応じて活用ください

②pictSpace

基本的な仕様は①のpictSQUARE-ONEと同様になります。違いは、他イベントへのショップ設定の引き継ぎが可能な点です。

今後もPICTSQUAREでの出展予定がある方にオススメです。

③その他通販サイト

名称の通り BOOTHやPixiv等の外部サービスのリンクを貼り、作品を公開する形式となります。サークルスペースから作品を閲覧する際に外部ページへの移動警告を挟みます。

一方でリンクは1つしか貼れない点に注意が必要です。
複数のサイトへのリンクを貼りたい場合は、プロカやLitLinlk等のリンクサイトに統合するなどの工夫が必要です。

サークル配置について

サークル配置について

  • 参加申込完了時に一度仮配置されますが、最終的には締切後に主催にて手動で再配置を行います。
  • 最終配置のお知らせは12月15日頃を予定しております。最終配置前のスペース番号等の告知はご遠慮ください。

その他の注意事項

事前入場について

  • サークル参加者は申請完了後、いつでも時間外入場が可能となります。
  • 事前入場時のスクリーンショットやSNSへの投稿はご自身のスペース及びアバターのみとさせていただきます。
    • ※他参加者のスペース・展示物等が写り込みのないよう配慮願います。
  • イベント当日までに自店舗から展示品のリンク先が見えるか確認ください。

イベント当日について

  • 開催当日はサークル主様が不在でも構いません。
  • 書き込みボードにサークル主様宛てのメッセージが書き込まれる場合があります。イベント終了後もMYページから確認できますので、ぜひ確認ください。
